I.T. Requirements

In I.T we required the following;
1.Application software and how to operate like Word,Excel spreadsheet,Powerpoint
2.Outlook Email messaging application
3.Internet,Definition usage and application.(Router setup and security firewalls specs)
4.Personal computer system components(Motherboards and peripherals)
5.Storage devices and power supplies(USB,Optical and hard drives form factor)
6.Understanding display devices(Monitors,l.c.d and projectors) and mobile displays.
7.Understanding Laptops and portable devices(differences and similarities)
8.Mobile devices operating system and application (Android OS and security management)
9.Installing and configuring printers.Current and old printers.
10.Operating features and interfaces(Windows OS ,Mac OS ,Linux)
11.Understanding networking.(cabling,trunking,naming and color coding)
12.Understanding network security fundamentals
13.System security(passwords,gateways,hacking resolved)
14.Basic coding in HTML and C.S.S (HTML is Hypertext Markup Language, a standardized
system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
And C.S.S is Cascading Style Sheet is a style sheet language used for describing the look and formatting of a document written in a markup language.)

Fashion & Design

For Fashion and Design, we had three main subjects that are the backbone of design. This are: • Creative Design
Learners get to learn how to put their creative ideas into paper and later present their designs in a neat and professional manner.
• Pattern making
Learners are taught how to translate designs into patterns which are then cut into fabric ready for sewing.
• Garment Construction
After Patterns are made they are then cut out into fabric ready to be stitched. Here learners are taught how to stitch together the different pattern pieces in a neat and professional manner resulting in a well made garment that is of a very high quality.

Careers that come with the following qualification
• Fashion illustration- Skills acquired in the Creative class are put into use here.
• Fashion designer- Responsible for coming up with new and unique designs using skills acquired in Creative Design class
• Pattern Maker- Responsible for translating designs into patterns using skills acquired in Pattern class
• Quality Assessor- Responsible for making sure that garments are made to the highest quality possible using skills acquired in Garment construction class
• Seamstress- Quality sewing of garment pieces together using skills acquired in Garment construction class
